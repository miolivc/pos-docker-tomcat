# pos-docker-tomcat
Aplicação que faz cadastro e listagem de pessoas.

#### 1. Executar a aplicação:
* Navegar até a pasta onde o download/clone foi feito e executar:  
    `sh run.sh`

#### 2. Parar a aplicação:
* Navegar até a pasta onde o download/clone foi feito e executar:  
    `sh stop.sh`

#### 3. Tecnologias usadas:
* Git
* docker
* HTML
* Java SE8
* Java EE7: Servlet
* Shellscript
